作品集后台说明
前台页面 无损原片页

Start Here

你只需要改一个文件

所有文字、个人信息、作品名称、视频链接，都在 `content.js` 里。改完点绿色按钮保存，网页会自动更新。

打开后台修改文件 查看面试官页面
01

打开后台

点上面的“打开后台修改文件”。如果 GitHub 让你登录，就用你注册的账号登录。

02

只改引号里的文字

例如把 `"你的姓名"` 改成你的真实姓名。不要删逗号、括号和冒号。

03

保存

页面右上角或底部找到绿色 `Commit changes`，点它，再点一次确认。等待 1 到 3 分钟。

What To Edit

具体改哪里

改个人信息

hero: {
  title: "你的姓名",
  role: "视频创作者 / 后期制作",
  copy: "你的个人介绍",
}

改作品标题

{
  id: "vlog-video-01",
  title: "Vlog 作品 01",
  short: "一句话介绍",
  description: "完整项目介绍",
}

填无损原片直链

media: {
  src: "网页预览版路径",
  losslessUrl: "https://这里粘贴原片直链"
}

关于无损视频，你记住这一句就够了

GitHub Pages 不能直接放 745MB、2.6GB 的原始 MOV。要无损，就把原片上传到“能提供公开 HTTPS 直链”的云存储，然后把直链填到 `losslessUrl`。
